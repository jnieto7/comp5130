TCPDUMP multi-capture

Capture of a simultaneous tcpdump to multiple hosts

Server Configuration

Capture options

Command Line:

-i <interface> Specifies the capture interface

-A Print frame payload in ASCII
-c <count> Exit after capturing count packets
-K Don't verify TCP checksums
-n Don't convert addresses to names
-s <len> Capture up to len bytes per packet


Capture Filter Primitives:

src host: Match IP source
dst host: Match IP destination
ether src host: Match ethernet source
ether dst host: Match ethernet destination
src port Match source port
dst port Match destination port
vlan: Match 802.1Q frames, optionally with a VLAN ID of vlan
