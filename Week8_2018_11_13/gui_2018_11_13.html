TCPDUMP multi-capture tool

This tool will allow the capture of a tcpdump to multiple hosts simultaneously

Server Configuration



Capture options

Command Line:

-A Print frame payload in ASCII
-c <count> Exit after capturing count packets
-e Print link-level headers
-F <file> Use file as the filter expression
-i <iface> Specifies the capture interface
-K Don't verify TCP checksums
-L List data link types for the interface
-n Don't convert addresses to names
-p Don't capture in promiscuous mode
-q Quick output
-s <len> Capture up to len bytes per packet
-S Print absolute TCP sequence numbers
-t Don't print timestamps
-v[v[v]] Print more verbose output


Capture Filter Primitives:

src host: Match IP source
dst host: Match IP destination
ether src host: Match ethernet source
ether dst host: Match ethernet destination
vlan: Match 802.1Q frames, optionally with a VLAN ID of vlan
